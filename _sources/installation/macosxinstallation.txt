Mac OS X Installation
---------------------

  1. Download the most recent version of SymPy `here <https://github.com/sympy/sympy/archive/master.zip>`_
  2. Unzip the zip file by right clicking the file and selecting "Extract All"
  3. Open up a command prompt in the directory the files were extracted and type ''python setup.py install''

If you have issues with this, please contact the
`PyDy Mailing list <http://groups.google.com/group/pydy>`_.
